Atalhos do ambiente de Desktop »
-----------------------------------------------
ATALHO                    DESCRIÇÃO

SUPER (Win) + 1..9              = Áreas de trabalho
SUPER (Win) + < >               = Foco na janela
ALT + Tab                       = Circular entre todas janelas
SUPER (Win) + Shift + Space     = Janela flutuante (toggle)
[APPLY] SUPER (Win) + r + < >   = Modo redimensionar janela
SUPER (Win) + Mouse Button 2    = Modo redimensionar janela (flutuante)
SUPER (Win) + f                 = Tela cheia
[APPLY] SUPER (Win) + w         = Agrupar em abas
SUPER (Win) + Mouse Button 1    = Arrasta janela (mouse)
SUPER (Win) + Shift + < >       = Mover janela
SUPER (Win) + q                 = Fechar janela
SUPER (Win) + Shift + 1..9      = Mover janela para a área de trabalho
Print                           = Capturar área selecionada
SUPER (Win) + Print             = Capturar janela em foco
SUPER (Win) + Shift + Print     = Capturar tela inteira
SUPER (Win) + g                 = Iniciar/Pausar/Retormar gravação de tela
SUPER (Win) + Shift + g         = Parar e salvar gravação de tela
SUPER (Win) + Shift + l         = Bloquear sistema
[APPLY] SUPER (Win) + Shift + m = Desligar monitor (não é suspender)
SUPER (Win) + Shift + e         = Sair do sistema
SUPER (Win) + Shift + r         = Recarregar Hyprland
